The Darwin setup based Carrol et al. 2020 Global Ecosystem has been created to simulate the impact of tDOC Mackenzie river runoff on the coastal Beaufort Sea

For this set up a new tDOC filed is created moving baseline setup from 31 to 32 passive tracers:
1. DIC     12. DOFe   23. c3
2. NO3     13. POC    24. c4
3. NO2     14. PON    25. c5
4. NH4     15. POP    26. c6
5. PO4     16. POFe   27. c7
6. FeT     17. POSi   28. Chl1
7. SiO2    18. PIC    29. Chl2
8. DOC     19. ALK    30. Chl3
9. tDOC    20. O2     31. Chl4
10. DON    21. c1     32. Chl5
11. DOP    22. c2      

The Ecosystem simulated contains (see Carrol et al., 2020):
  > 5 phytoplankton types:
      - Diatoms
      - Large Eukaryotes
      - Synechococcus type 1
      - Synechococcus type 2
      - Prochlorococcus
  > 2 Zooplankton types:
      - Small zooplankton (will feed on pico-euk, and bacteria)
        nominal size 45um ESD
      - Large zooplankton (will feed on diatoms, haptophyte and dinoflagellate)
        nominal size 100um
